# Accounting App

A cross-platform accounting app built with Ionic and Angular using the MVVM architecture. It features:

- Sales invoices
- Item management
- Error handling
- Authentication
- Role-based permissions
- Robust routing





As part of the project, we also built dedicated backup offices, with the ability to securely upload data to Google Drive using Java Native techniques, ensuring data protection and easy retrieval.
